# Features Overview

**Whiteboard Video Maker bundles a hand-drawn animation editor, a keyframe timeline, built-in narration, and client-side video export into one free web + mobile app.** This page is the map; each row links to a deeper page.

## The canvas & board

| Capability | What it does |
|---|---|
| [Board styles](./board-styles.md) | Whiteboard, blackboard, glassboard, and greenscreen backgrounds. |
| Multi-layer canvas | A Konva-powered canvas with layers you can order, hide, and lock. |
| Snap & alignment | Snap-to-grid and alignment guides while you place elements. |

## Drawing & content

| Capability | What it does |
|---|---|
| [Drawing tools](./drawing-tools.md) | Pen/freehand, shapes, arrows, text, and images with hand-drawn sketch styling. |
| Text effects | Typewriter and fade reveals for text elements. |
| Shape morphing | Interpolate one shape into another over time. |
| Path editor | Draw and edit custom motion/draw paths. |
| Color filters | Apply color adjustments to elements. |
| Particles | Add particle effects for emphasis. |

## Motion & timing

| Capability | What it does |
|---|---|
| [Timeline & keyframes](./timeline-keyframes.md) | Layered timeline with per-element keyframes and animation curves. |
| Draw order | Control the exact order the hand reveals elements. |
| Transitions | Scene/element transitions between segments. |
| Hand animation | A follow-along drawing hand that traces strokes. |
| Smart timing & beat markers | Timing presets and beat-aligned reveals. |
| Loop markers | Mark loopable segments. |

## Audio & narration

| Capability | What it does |
|---|---|
| [Text-to-speech](./audio-narration.md) | Voiceovers from the browser Web Speech API (no paid TTS). |
| Voice recording | Record your own narration in-app. |
| Audio mixer & trimmer | Layer, trim, and balance audio tracks. |
| Sound effects | Drop in sound effects. |
| Synced text animation | Animate text in sync with narration. |

## Assets

| Capability | What it does |
|---|---|
| [Asset libraries](./assets-libraries.md) | Icon, character, image, and emoji libraries. |
| Fonts | Bundled font choices (Inter, Poppins, Montserrat, Open Sans, Roboto). |
| My Assets | Your own uploads (stored via FilesHub). |
| Element suggestions | Suggested elements to add. |

## Projects & export

| Capability | What it does |
|---|---|
| [Save & sync](../guides/save-and-sync-projects.md) | Cloud projects + version history when signed in. |
| Templates | Save and reuse project templates. |
| [Export](../guides/export-webm-gif.md) | WebM video and animated GIF, rendered client-side. |
| Frame export | Export individual frames or a frame sequence. |
| Embed & share | Generate an embed snippet and share via the native share sheet or web modal. |

## Cost & access

Free, no usage caps, no watermark. Web works signed-out; signing in adds saving, sync, and version history.
