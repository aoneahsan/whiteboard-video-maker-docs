# Assets & Libraries

**You don't have to draw everything from scratch — Whiteboard Video Maker includes icon, character, image, and emoji libraries, plus a place for your own uploads.**

## Built-in libraries

| Library | What's in it |
|---|---|
| **Icon library** | A searchable set of icons to drop onto the board. |
| **Character library** | Characters/figures for explainer scenes. |
| **Image library** | Ready-to-use images. |
| **Emoji picker** | Insert emoji as elements. |
| **Asset library** | A general catalog of reusable elements. |

Each library item is added to the canvas as a normal element, so it animates on the timeline like anything you draw.

## Fonts

Text elements can use the bundled fonts: **Inter, Poppins, Montserrat, Open Sans, and Roboto.** Pick one per text element from the font picker.

## Your own uploads — "My Assets"

Upload your own images and reuse them across projects. Uploaded files are stored through **FilesHub** (the project's free file-storage backend), not on a per-project server. When you delete an asset, it is removed from storage too.

> **Note:** uploading and reusing your own assets is tied to your account so your library follows you across devices.

## Element suggestions

The editor can suggest elements to add based on your scene, helping you fill out a layout quickly.
