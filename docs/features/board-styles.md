# Board Styles

**A board style sets the background and the default stroke contrast for your animation.** Whiteboard Video Maker ships four, and you can switch between them at any time without redrawing your elements.

## The four styles

| Style | Background | Best for |
|---|---|---|
| **Whiteboard** | Light/white | The classic explainer look; dark sketch strokes on white. |
| **Blackboard** | Dark/chalk | Chalk-style lessons; light strokes on a dark surface. |
| **Glassboard** | Translucent/tinted | A modern, "drawing on glass" aesthetic facing the viewer. |
| **Greenscreen** | Solid green | Recording over a key color so you can composite the animation onto other footage later. |

## When to use greenscreen

Pick **greenscreen** when you intend to overlay the animation onto a video in a separate editor. Export the WebM, then chroma-key the green background out in your video tool. Because the export is a standard WebM, any editor that supports chroma key will work.

## Switching styles

Changing the board style only changes the background and default contrast — your elements, timeline, and keyframes stay intact. If you switch from whiteboard to blackboard, adjust element colors so strokes remain readable against the new background.

## Tips

- Use **whiteboard** for the widest reuse — it reads well on every platform.
- Use **blackboard** for a focused, lecture feel.
- Keep text high-contrast against the board; check it in the [preview](../guides/create-first-video.md) before exporting.
